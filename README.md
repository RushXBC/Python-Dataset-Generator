Synthetic Data Generators

This repository contains Python scripts for creating synthetic datasets for testing, analytics, or simulation purposes. Each script is designed to generate realistic, structured data that can be used for dashboards, analytics, machine learning, or other experiments.

Features:
* Customizable data structures – Define your own entities, attributes, and relationships.
* Multi-dimensional simulations – Generate data for multiple categories, segments, or groups.
* Time-based or event-based simulation – Daily, monthly, or custom time intervals with seasonal trends.
* Behavioral logic – Incorporate realistic patterns, probabilities, or conditional rules.
* Inventory, transactions, or event tracking – Optional tracking of quantities, states, or statuses.
* Output-ready CSVs – Easy to use in Excel, Tableau, Python, or other tools.

Included Scripts:
Each script is independent and generates its own dataset. Examples include:

* Shoe Store Sales Generator – Simulates daily orders, discounts, and stock levels for multiple brands.
* (Future scripts can include other domains, e.g., hospital patient records, e-commerce transactions, sensor data, etc.)
